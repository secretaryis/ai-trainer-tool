# دليل المستخدم (AR)

1. شغّل التطبيق (`python src/main.py`).
2. من تبويب "المعالج" اتبع الخطوات: اختيار نموذج صغير، إدخال البيانات، بدء التدريب، التصدير، ثم تشغيل عبر Ollama.
3. التبويبات المتقدمة: العتاد والإعدادات، النموذج والبيانات، التدريب، الاختبار والتصدير، Ollama.
4. صيغ التصدير: PyTorch، SafeTensors، ONNX، و GGUF (لـ Ollama).
5. استكشاف الأخطاء: تأكد من توفر الإنترنت، ووجود مساحة قرص كافية (2GB على الأقل) قبل التصدير.
